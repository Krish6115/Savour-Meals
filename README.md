# Saviour-Meals

Saviour-Meals is an awesome platform designed to connect food donors with NGOs and volunteers, facilitating the redistribution of excess food to those in need.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- Complete Food Donation Management System
- Segregated Dashboards for Donors, NGOs, and Volunteers
- Real-time location tracking for food pickups
- OTP-based secure food delivery verification

## Getting Started

1. Set up the environment variables (`.env` files in both client and server)
2. `cd server && npm install && npm run dev`
3. `cd client && npm install && npm start`
